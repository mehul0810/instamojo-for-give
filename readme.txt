== Installation ==
= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don\'t need to leave your web browser. To do an automatic install of Instamojo for GiveWP, log in to your WordPress dashboard, navigate to the Plugins menu and click \"Add New\".

In the search field type \"Instamojo for GiveWP\" and click Search Plugins. Once you have found the plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking \"Install Now\".

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
